# Projects Overview

This portfolio showcases open-source and personal projects by Ziv Hochman. Key themes include machine learning, interactive visualization, full-stack web development, and human-centered AI.

## Interactive AI Portfolio

A conversational portfolio where visitors chat with an assistant grounded in authentic writings about experience, research, and projects. Built with React, Vite, Tailwind, FastAPI, RAG over personal knowledge, and Ollama for inference.

## Research & Data Science

Work spans interpretable ML, visual analytics, and applied data science—including MSc research at Utrecht University on human-centered AI for real-world decision-making.

## Tech Stack Highlights

- **Languages:** Python, TypeScript/JavaScript, SQL
- **Frontend:** React, Vite, D3.js, Framer Motion
- **ML/AI:** scikit-learn, PyTorch ecosystem tools, RAG pipelines, local LLMs via Ollama
- **Infrastructure:** FastAPI, PostgreSQL with pgvector, Redis, Fly.io
